# ColorCorrection

Репозиторий посвящён воспроизведению и расширению результатов статьи **A. Kucuk, G. D. Finlayson, R. Mantiuk, M. Ashraf, “Performance Comparison of Classical Methods and Neural Networks for Colour Correction” (2023)**.

Основная цель проекта — реализовать классические методы цветокоррекции, повторить результаты **Table 2** из статьи по метрике **CIE Delta E 2000**, а затем расширить регрессионный подход более мощными признаковыми моделями и улучшенной процедурой оптимизации.

---

## Что делает проект

Проект строит отображение

$\mathbf p = (R,G,B)^\top \mapsto \hat{\mathbf x} = (X,Y,Z)^\top$



где отклик камеры переводится в эталонное пространство **CIE XYZ**, после чего качество оценивается в **CIELAB** по метрике

$
\Delta E_{2000}(Lab(\hat{\mathbf x}), Lab(\mathbf x)).
$

Используемый вычислительный конвейер:

1. загрузка спектральных данных: отражательные способности, спектральные чувствительности камеры, освещение D65 и функции цветового соответствия CIE 1931;
2. численное интегрирование спектров для получения пар **RGB / XYZ**;
3. обучение модели цветокоррекции;
4. оценка по **5-fold cross-validation**;
5. расчёт статистик ошибки: **Mean**, **Max**, **Median**, **95%**.

---

## Реализованные методы

### Базовые методы из статьи

- **LS** — линейная регрессия;
- **LS-P** — полиномиальная регрессия второго порядка;
- **LS-RP** — root-polynomial регрессия второго порядка;
- **LS-Opt** — оптимизация линейной модели по perceptual-loss;
- **LS-P-Opt** — оптимизационная версия polynomial-регрессии;
- **LS-RP-Opt** — оптимизационная версия root-polynomial-регрессии.

### Расширения проекта

В рамках проекта были добавлены новые варианты, которых нет в статье:

- **LS-RP3** — root-polynomial признаки 3-го порядка;
- **LS-RP4** — root-polynomial признаки 4-го порядка;
- **LS-RP3-Opt** — оптимизационная версия LS-RP3;
- **LS-RP4-Opt** — оптимизационная версия LS-RP4.

Идея расширения состоит в переходе от базиса второго порядка

$
\phi_{RP2}(R,G,B)=\bigl(R,G,B,\sqrt{RG},\sqrt{RB},\sqrt{GB}\bigr)^\top
$

к более богатым root-polynomial признакам 3-го и 4-го порядка, например

$
(R^2G)^{1/3},\quad (RGB)^{1/3},\quad (R^3G)^{1/4},\quad (R^2GB)^{1/4}.
$

Такие признаки сохраняют однородность по экспозиции:

$
\phi(\alpha R,\alpha G,\alpha B)=\alpha\,\phi(R,G,B),\qquad \alpha>0.
$

## Что было улучшено по сравнению со статьёй

Помимо расширения признакового пространства, в проект были добавлены улучшения процедуры обучения:

- **multi-start** для `-Opt` методов: оптимизация запускается из нескольких начальных точек;
- **ridge-регуляризация**;
- **автоматический подбор** параметра регуляризации $\lambda$ через **nested cross-validation**;
- возможность оптимизировать модель не только по ошибке в **CIELAB**, но и напрямую по **CIEDE2000**.

Оптимизационная постановка имеет вид

$
M^*=\arg\min_M \Bigl(\mathcal L(M)+\lambda\|M\|_F^2\Bigr),
$

где $\mathcal L$ — либо ошибка в пространстве Lab, либо средняя perceptual-ошибка $\Delta E_{2000}$.

- `main.py` — CLI-интерфейс и запуск экспериментов;

## Как запускать

### Базовые методы

```bash
make run-ls
make run-lsp
make run-lsrp
make run-lsopt
make run-lspopt
make run-lsrpopt
```

```bash

make run-lsrp3opt OPT_STARTS=10 AUTO_RIDGE=1 SHOW_SELECTED_RIDGES=1
```

```bash
make run-lsrp3opt-de2000 OPT_STARTS=10 AUTO_RIDGE=1
```

## Ключевые наблюдения

1. Базовые методы **LS / LS-P / LS-RP / LS-Opt / LS-P-Opt / LS-RP-Opt** воспроизводятся очень близко к статье.
2. Небольшие расхождения со статьёй, вероятнее всего, связаны с:
   - иным разбиением в **5-fold cross-validation**;
   - чувствительностью статистик **Max** и **95%** к выбросам;
   - локальным характером оптимизации **Nelder–Mead**;
   - мелкими отличиями численной реализации спектрального препроцессинга.
3. Расширенные методы **LS-RP3**, **LS-RP4**, **LS-RP3-Opt**, **LS-RP4-Opt** показывают лучший результат, чем исходные классические методы из статьи.
4. Лучший результат в текущем проекте по среднему значению ошибки достигается у **LS-RP4** и **LS-RP4-Opt**:

   $
   \text{Mean}=0.64.
   $

5. Лучший хвост распределения ошибок среди расширенных методов достигается у **LS-RP3-Opt** и **LS-RP4-Opt**:

   $
      95\%=1.94.
   $

## Вывод

Добавление **root-polynomial признаков 3-го и 4-го порядка**, **multi-start оптимизации** и **ridge-регуляризации с nested CV** позволило получить более гибкие и более точные регрессионные модели.

Практический итог проекта: **классический регрессионный подход после аккуратного расширения признаков и оптимизации остаётся крайне конкурентоспособным и способен превосходить базовые результаты статьи без использования нейросети**.
