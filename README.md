# ColorCorrection

Отборочный проект в **ИППИ РАН**: небольшой набор инструментов для экспериментов с методами **цветокоррекции камеры/дисплея**.

## Что внутри
- Базовые решения и эксперименты (добавлю новые методы)
- Утилиты для работы со спектральными данными и метриками качества

## Запуск
1) Установка зависимостей:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install numpy pandas
```

2) Размещаем локальные файлы данных на диск (они игнорируются git), затем запусти основной скрипт:
```bash
python3 main.py
```

3) Если нужно запустить обучение базовой LSQ-модели напрямую:
```bash
python3 train_lsq.py
```

## Данные
Крупные датасеты не хранятся в репозитории (см. `.gitignore`). Данные храни локально и передавай пути через параметры командной строки.
